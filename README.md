# MaterialDesign

![YpyKH.png](https://s1.ax1x.com/2017/10/17/YpyKH.png)

# Material Design Features

- Button Alpha Effect
- Button Ripple Effect
- Card View
- Text Input 
- Switch
- Check Box
- Radio Button
- Floating Action Button
- Date Picker
- Circular Reveal Animation
- Bottom Navigation Bar (more than 4 items disable shift mode)
- Collapse Tool Bar
- Snackbar
